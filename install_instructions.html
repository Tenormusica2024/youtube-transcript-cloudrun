🎬 YouTube字幕抽出ツール

YouTube動画の字幕を自動抽出・AI要約する簡単インストール

🥇 Tampermonkey版（最も簡単） 推奨 30秒

最も簡単で高速な配布方法。1クリックで自動インストール可能。

前提条件: Tampermonkey拡張機能（無料）
  1. Tampermonkey拡張機能をインストール（初回のみ）
  2. 下記の「ワンクリックインストール」ボタンをクリック
  3. インストール確認画面で「インストール」をクリック
  4. YouTubeで動画を開くと字幕抽出ボタンが自動表示
1. Tampermonkey インストール 2. ワンクリックインストール

🥈 Chrome拡張機能版（高機能） 1分

フル機能版。ポップアップUI、設定管理、通知機能を含む完全版。

  1. ZIPファイルをダウンロード・展開
  2. Chrome設定で「拡張機能」→「デベロッパーモード」有効化
  3. 「パッケージ化されていない拡張機能を読み込む」で展開フォルダを選択
  4. YouTubeで動画を開くと字幕抽出ボタンが表示
Chrome拡張機能をダウンロード 詳細な手順を確認

🥉 CRXダイレクト版（中級者向け） 1分

CRXファイルを直接インストール。技術知識が必要ですが高速。

  1. CRXファイルをダウンロード
  2. Chrome拡張機能ページ（chrome://extensions/）を開く
  3. CRXファイルをページにドラッグ＆ドロップ
  4. 「拡張機能を追加」で確認

⚙️ サーバー設定（インストール後）

インストール後、字幕抽出サーバーのURLを設定してください：

  • ローカル開発: http://localhost:8085
  • ngrokトンネル: https://xxx.ngrok-free.app
  • 本番環境: https://your-domain.com
Tampermonkey版: 拡張機能メニュー → 「⚙️ サーバーURL設定」
Chrome拡張機能版: 拡張機能アイコンクリック → サーバーURL入力
