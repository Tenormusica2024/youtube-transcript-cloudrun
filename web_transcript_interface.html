🎬 YouTube字幕抽出

GitHub Actions経由でIPブロック回避

⚠️ 注意: この方法はCloud RunのIPブロック問題を回避するための迂回手段です。
🔧 方法1: GitHub Actions自動実行
GitHub Actionsでスクリプトを実行し、結果をGitHub Pagesで表示
📝 手順:
  1. 上記のURLと言語を設定してボタンをクリック
  2. GitHubリポジトリが開きます
  3. Actions > Manual Transcript Extraction を選択
  4. Run workflow でパラメータを入力
  5. 数分後に結果がPages URLに表示されます
📂 GitHubリポジトリを開く
☁️ 方法2: GitHub Codespaces
ブラウザ上のVS Codeで直接Pythonスクリプトを実行
📝 手順:
  1. 上記ボタンでCodespacesが開きます
  2. ターミナルで python local_transcript_extractor.py を実行
  3. YouTube URLを入力してEnter
  4. 数分後にテキストファイルが生成されます
  5. ファイルエクスプローラーからダウンロード可能
🌐 方法3: Replit Online IDE
Replitでオンライン実行環境を利用
📝 手順:
  1. 上記ボタンでReplitが開きます
  2. Runボタンでスクリプト実行
  3. コンソールでYouTube URLを入力
  4. 結果がファイルに保存されます
  5. Filesタブからダウンロード
🧩 方法4: ブラウザ拡張機能
YouTube画面に直接字幕抽出ボタンを追加
🚧 開発中の機能:

YouTubeの動画ページに「字幕抽出」ボタンを追加する拡張機能を開発中です。

完成すると、YouTube動画を見ながらワンクリックで字幕を抽出できるようになります。

✅ 処理完了

結果がGitHub Pages または一時ファイルに保存されました。
